2 Installation of the GradedRingForHomalg Package

To install this package just extract the package's archive file to the GAP pkg directory. GradedRingForHomalg also needs the package homalg.

By default the GradedRingForHomalg package is not automatically loaded by GAP when it is installed. You must load the package with

LoadPackage("GradedRingForHomalg");

before its functions become available.
