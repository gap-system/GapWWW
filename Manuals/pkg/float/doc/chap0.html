Abstract

This document describes the package Float, which implements in GAP arbitrary-precision floating-point numbers.

The computer algebra system GAP is available at http://gap-system.org.

This documentation for Float is available at http://www.gap-system.org/Manuals/pkg/float-0.9.1/doc/manual.pdf in PDF format, and may be accessed online at http://gap-packages.github.io/float/.

The latest release of the package may be downloaded as https://github.com/gap-packages/float/releases/download/v0.9.1/float-0.9.1.tar.gz (tar, gzipped). The latest repository version (possibly unstable) may be downloaded as https://github.com/gap-packages/float/tarball/master (tar, gzipped), https://github.com/gap-packages/float.git (git repository), https://packages.debian.org/search?keywords=gap-float&searchon=names&suite=all§ion=all (debian package), or explored at https://github.com/gap-packages/float/tree/master/.

For comments or questions on Float please contact the author.
