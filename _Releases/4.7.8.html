Linux and macOS

Download one of the archives below, unpack it and run ./configure; make in the unpacked directory. Then put and run in the pkg subdirectory the shell script InstPackages.sh (for 32-bit builds on 64-bit machines, use InstPackages32.sh) in order to build most of the packages that require compilation provided sufficiently many libraries, headers and tools are available (you may have to run chmode first). For further details, see here. Expert users can find the desciption of all installation options in the INSTALL file.

Windows

Use the .exe installer which contains binaries for GAP and some packages and provides the standard installation procedure. Note that the path to the GAP directory should not contain spaces. For example, you may install it in C:\gap4r7 (default), D:\gap4r7p6 or C:\Math\GAP\gap4r7, but you must not install it in a directory named like C:\Program files\gap4r7 or C:\Users\alice\My Documents\gap4r7 etc.
