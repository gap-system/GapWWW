This library is part of the core GAP system.

Authors

This library has been built up by various people in the course of the development of GAP.

Contents

The data library of basic groups provides generic functions to construct

  • cyclic groups,
  • abelian groups,
  • elementary abelian groups,
  • dihedral groups,
  • extraspecial groups,
  • alternating groups,
  • symmetric groups,
  • Mathieu groups,
  • Suzuki groups, and
  • Ree groups.
