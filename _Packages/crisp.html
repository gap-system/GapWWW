In particular, it allows to compute C-radicals and C-injectors for Fitting classes (and Fitting sets) C, C-residuals for formations C, and C-projectors for Schunck classes C. In order to carry out these computations, the group class C must be represented by an algorithm which can decide membership in the group class.

Moreover, CRISP contains algorithms for the computation of normal subgroups invariant under a prescribed set of automorphisms and belonging to a given group class.

This includes an improved method to compute the set of all normal subgroups of a finite soluble group, its characteristic subgroups, minimal normal subgroups and the socle and p-socles for given primes p. status: accepted communicated-by: "Joachim Neubüser (Aachen)" accept-date: "12/2000" source-repo: type: git url: https://github.com/bh11/crisp.git issue-url: "https://github.com/bh11/crisp/issues" support-email: doc-html: htm/chapters.htm doc-pdf: doc/manual.pdf keywords: | radical, residual, projector, injector, formation, Fitting class, Fitting set, Schunck class, group class, normal subgropus, minimal normal subgropus, socle. citeas: |
