All the functions in this package are supposed to be capsuled by functions in the Convex package, which provides needed structures and datatypes. Also the functions the have nicer names. This fact also causes that there are no doumentations for functions in this package. To get an overview about the supported functions, one might look at the polymake_main.cpp file or simply message the author.
