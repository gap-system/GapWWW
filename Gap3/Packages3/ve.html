Vector Enumeration, linear Todd-Coxeter method

Share package since release 3.4, about May 1994.
This package has not yet been transferred to or replaced in GAP 4. You can use it only in GAP 3.

Author

Steve Linton.

Implementation

Language: C
Operating system: Unix
Current version: 3 (in the 3.4.4 distribution)

Description

The VE package provides access to the implementation of the "linear Todd-Coxeter" method for computing matrix representations of finitely presented algebras.

Manual

A VE manual is given in chapter 72 of the GAP 3 manual.
