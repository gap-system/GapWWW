GRIM: Groups of Rational and Integer Matrices

Share package since release 3.4, about April 1997, communicated by Akos Seress (1958-2013).
This package has not been transferred to or replaced in GAP 4, but some of its functionality, e. g. determining whether an integral matrix group is finite, is available (by a different algorithm) as a method for 'IsFinite' in GAP 4.

Author

Robert Beals.

Implementation

Language: GAP 3
Operating system: Any
Current version: 1.0 (in the 3.4.4 distribution)

Description

GRIM (Version~1.0) is a system for testing finiteness and integrality of rational matrix groups.

Manual

A GRIM manual is given in chapter 64 of the GAP 3 manual.
