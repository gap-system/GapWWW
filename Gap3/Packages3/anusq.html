ANU Soluble Quotient algorithm for finitely presented groups

Share package since release 3.4, about April 1994.
This package has not yet been transferred to or replaced in GAP 4. You can use it only in GAP 3.

Author

Alice Niemeyer.

Implementation

Language: C
Operating system: Unix
Current version: 1.0 (in the 3.4.4 distribution)

Description

The function Sq is the interface to a Soluble Quotient standalone program.

Sq computes a consistent power conjugate presentation for a finite soluble group given as a quotient of a finitely presented group G.

Manual

An ANU SQ manual is given in chapter 56, section 4 of the GAP 3 manual.
