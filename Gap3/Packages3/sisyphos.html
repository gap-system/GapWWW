SISYPHOS: p-groups and their modular group algebras

Share package since release 3.3, about October 1993.
This package has not been transferred to GAP 4. However, its functionality is mainly covered by the GAP 4 package LAGUNA.

Author

Martin Wursthorn.

Implementation

Language: C
Operating system: Unix
Current version: 0.8.3 (available here)

Description

SISYPHOS provides access to implementations of algorithms for dealing with p-groups and their modular group algebras. At the moment only the programs for p-groups are accessible via GAP 3. They can be used to compute isomorphisms between p-groups, and automorphism groups of p-groups.

Manual

A Sisyphos manual is given in chapter 70 of the GAP 3 manual.
