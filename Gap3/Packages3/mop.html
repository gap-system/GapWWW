Action of parabolic subgroups on the Lie algebra of the unipotent radical

Share package since December 2001, communicated by Gerhard Hiss, Aachen.
This package has not yet been transferred to or replaced in GAP 4. You can use it only in GAP 3.

Authors

Ulf Juergens and {% include namelink.html manual="Gerhard Erich Roehrle" text="Gerhard Erich Roehrle" %}.

Implementation

Language: GAP 3
Operating system: Any
Current version: 1.0 (in the 3.4.4 distribution)

Description

MOP is designed to compute upper bounds for the modality of the action of parabolic subgroups P on pu, the Lie algebra of the unipotent radical of P, for simple algebraic groups G over an algebraically closed field K. It is a GAP 3 share package based on the CHEVIE package.
