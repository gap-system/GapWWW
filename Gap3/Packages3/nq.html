Nilpotent Quotient algorithm for finitely presented groups

Share package since release 3.2, about January 1993.
Corresponding GAP 4 package: GAP 4 package "nq".

Author

Werner Nickel.

Implementation

Language: C
Operating system: Unix
Current version: (no version number given in the 3.4.4 distribution)

Description

The NQ package computes the quotient groups of a finitely presented group F successively modulo the terms of the lower central series of F. If it terminates, it returns a list L. The i-th entry of L contains the non-trivial abelian invariants of the i-th factor of the lower central series of F (the largest abelian quotient being the first factor).

Manual

An NQ manual is given in chapter 56, section 10 of the GAP 3 manual.
