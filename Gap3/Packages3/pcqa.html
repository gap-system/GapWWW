Polycyclic quotient algorithm for finitely presented groups

Share package since release 3.4, about April 1997, communicated by Derek F. Holt, Warwick.
This package has not yet been transferred to or replaced in GAP 4. You can use it only in GAP 3.

Author

Eddie Lo.

Implementation

Language: C

Current version: (no version number given in the 3.4.4 distribution)

Description

This package is written by Eddie Lo. The original program is available for anonymous ftp at math.rutgers.edu. The program is an implementation of the Baumslag-Cannonito-Miller polycyclic quotient algorithm and is written in C.

Manual

A PCQA manual is given in chapter 69 of the GAP 3 manual.
