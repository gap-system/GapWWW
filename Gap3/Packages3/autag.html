AutAG: Automorphism groups of finite solvable groups

Share package since release 3.4, about November 1996, communicated by Charles Wright, Eugene.
This package has not been transferred to GAP 4. However, a similar algorithm is implemented in the main GAP 4 library and it is available via the operation 'AutomorphismGroup'.

Author

Michael J. Smith.

Implementation

Language: GAP 3
Operating system: Any
Current version: 1.0 (in the 3.4.4 distribution)

Description

A package for calculating the automorphism group of a finite solvable group given as special ag group. Generators of the automorphism group exhibit a normal series of the automorphism group.

Manual

An AutAG manual is given in chapter 58 of the GAP 3 manual.
