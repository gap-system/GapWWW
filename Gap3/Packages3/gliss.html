GLISSANDO: Small Semigroups and Nearrings

Share package since release 3.4, about Apri 1997.
This package has not been transfered to GAP 4, but the GAP 4 package SONATA covers its functionality related to nearrings, and a library of small semigroups in GAP 4 is Smallsemi.

Author

Christof Noebauer.

Implementation

Language: C
Operating system: Any
Current version: 1.0 (in the 3.4.4 distribution)

Description

GLISSANDO is a GAP 3 package consisting of libraries of small semigoups (orders 1 - 5) and near-rings (orders 2 - 15) and functions for examining them.

Manual

A GLISSANDO manual is given in chapter 62 of the GAP 3 manual.
