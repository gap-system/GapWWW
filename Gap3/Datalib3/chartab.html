Relation to the corresponding GAP 4 data library

The GAP 4 Data Library Character Tables is a considerable extension of the GAP 3 library and technically has become a GAP 4 package. All GAP 3 functions involving characters will work with this GAP 4 library (see the home page The GAP Character Table Library for details).

Contents

All character tables in the ATLAS and the Modular ATLAS with many additions.
