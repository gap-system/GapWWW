Isabel Araújo and Andrew Solomon.

Implementation

Language:
Operating system:
Download: commsemi.tar.gz

Description

CommSemi provides special algorithms for commutative semigroups using the RequirePackage interface. Extract the file commsemi.tar.gz into your pkg directory - no other setup is required.

Further information on usage of Commsemi and other semigroups functionality in GAP can be found in the authors' "Tutorial - Computing with semigroups in GAP", which is available from http://www.illywhacker.net/index.php?content=papers
